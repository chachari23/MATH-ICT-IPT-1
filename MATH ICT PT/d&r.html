Domain And Range

A relation can only be a function if and only if there are no repeating x-coordinates or domain. Each abscissa must be unique in a set of ordered pairs.

Using the mapping diagram, there are three representations of relations, namely one-to-many one-to-one, many-to-one, and one-to-many.

Examples

One-to-one

The set of ordered pairs in the given map is as follows: (2, a), (4,b), (6,c), (8,d).

Each X-coordinate is paired with a unique y-coordinate, which gives a one-to-one relationship. There is no repeated domain, and then this is a function.



Many-to-one

The set of ordered pairs in the given map is as follows: {(3,2), (4,2), (9,2)}

The X-coordinates are paired with a single y-coordinate Each abscissa is unique, hence, this is a function.



One-to-many

The set of ordered pairs given map is as follows: {(a,x), (a,y), (a,z)}

In this case, the x-coordinate is pained with three different y-coordinates making the abscissa not unique. This is just a relation and not a function.
